## XM\_GET\_CHANNEL\_DETAILS\_CHANGED

Displays current channel details.


### Name

`XM_GET_CHANNEL_DETAILS_CHANGED ()`


| Parameter | Type | Description      |
| --------- | ---- | ---------------- |
| CHANNEL   | INT  | Channel          |
| CATEGORY  | STR  | Channel category |
| NAME      | STR  | Channel name     |


### Returns

`None`
